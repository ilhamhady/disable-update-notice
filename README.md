# Disable Update Notices

A lightweight WordPress plugin that allows you to selectively disable update notices for plugins, themes, and WordPress core.

## Features

- **Plugin Updates** — Hide update notices for selected plugins
- **Theme Updates** — Hide update notices for selected themes
- **WordPress Core Updates** — Disable core, major, minor, and translation update notices individually

## Installation

1. Download or clone this repository into your `wp-content/plugins/` directory
2. Activate the plugin through the **Plugins** menu in WordPress
3. Go to **Settings > Disable Update Notices** to configure

## Usage

Navigate to **Settings > Disable Update Notices** where you'll find three tabs:

- **Plugin Updates** — Check the plugins you want to hide update notices for
- **Theme Updates** — Check the themes you want to hide update notices for
- **WordPress Updates** — Toggle core, major, minor, and translation update notices

Select the items you want to suppress and click **Save Changes**.

## Requirements

- WordPress 5.0 or higher
- PHP 7.0 or higher
